### 컬러 팔레트 설계 (5종)

에듀테크 브랜드 아이덴티티에 맞는 5종 컬러 팔레트를 설계하였다.

| 이름 | 역할 | 기준색 (400) |
|------|------|------------|
| **blush** | 베이비 핑크 — 메인 컬러 | `#F4A0B9` |
| **azure** | 베이비 블루 — 메인 컬러 | `#60ABE5` |
| **sage** | 민트/세이지 — 보조 컬러 | `#33BC9A` |
| **plum** | 라벤더/퍼플 — 보조 컬러 | `#A882F2` |
| **coral** | 포인트 컬러 — CTA/강조 | `#FF6B7A` |

각 컬러는 50~900 단계의 명도 스케일을 가지며 `tailwind.config.js`에 등록하였다.

### 다크모드 / 라이트모드 구현

- `tailwind.config.js`: `darkMode: 'class'` 설정
- `useDarkMode` 커스텀 훅: `localStorage` 기반 상태 영속화
- `index.html` 인라인 스크립트: 첫 렌더 전 다크클래스 적용 (FOUC 방지)
- 헤더 우측 Sun/Moon 아이콘 토글 버튼

### 페이지 구성

| 경로 | 페이지 | 주요 내용 |
|------|--------|----------|
| `/` | 홈 | 히어로 슬라이더, 통계, 특징, 과목 카테고리, 프로그램, CTA |
| `/videos/:subject` | 동영상 강의 | 과목별 탭, 2×3 유튜브 그리드, 페이지네이션 |
| `/workbook/:subject` | AI 문제집 | 과목별 탭, 난이도별 문제집 카드, AI 생성기 배너 |
| `/programs` | 교육 프로그램 | 4대 프로그램 소개, 이용 단계, 과목별 바로가기 |
| `/about/:tab` | 소개 | 회사소개 / 비전 / 연혁 탭 |
| `/guide` | 이용 안내 | 이용 방법, 요금제, FAQ, 문의 |

### 동영상 강의 페이지 상세

- **2열 × 3행 고정 그리드** (모바일: 1열)
- 페이지당 6개 영상, 페이지네이션으로 이동
- YouTube 썸네일 클릭 → iframe 인라인 재생
- youtubeId가 `YOUTUBE_ID_HERE`인 경우 플레이스홀더 카드 표시
- 영상은 YouTube **링크 공개(Unlisted)** 설정 권장 (비공개 embed 불가)
