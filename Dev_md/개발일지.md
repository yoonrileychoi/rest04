### 컬러 팔레트 설계 (5종)

에듀테크 브랜드 아이덴티티에 맞는 5종 컬러 팔레트를 설계하였다.

| 이름 | 역할 | 기준색 (400) |
|------|------|------------|
| **blush** | 베이비 핑크 — 메인 컬러 | `#F4A0B9` |
| **azure** | 베이비 블루 — 메인 컬러 | `#60ABE5` |
| **sage** | 민트/세이지 — 보조 컬러 | `#33BC9A` |
| **plum** | 라벤더/퍼플 — 보조 컬러 | `#A882F2` |
| **coral** | 포인트 컬러 — CTA/강조 | `#FF6B7A` |

각 컬러는 50~900 단계의 명도 스케일을 가지며 `tailwind.config.js`에 등록하였다.

### 테마 전환 기능 (컬러 팔레트 스위처)

헤더 우측에 5개 컬러 스와치 칩을 배치하여 클릭 시 사이트 전체 색상이 실시간으로 변경된다.

- `useTheme` 커스텀 훅: CSS 사용자 정의 변수(`--t-from`, `--t-to` 등)를 `document.documentElement`에 동적 적용
- `localStorage`에 선택 테마 저장 → 새로고침 후에도 유지
- `btn-primary`, `tab-btn.active`, 스크롤 버튼 등 주요 UI 요소가 테마 변수를 따름
- 선택된 스와치에 링(ring) 표시로 현재 테마 강조

| 테마 | 그라데이션 | hex |
|------|-----------|-----|
| 핑크 (기본) | blush → coral | `#EC7A9A` → `#FF6B7A` |
| 블루 | azure → plum | `#3892D8` → `#8B58E8` |
| 민트 | sage → azure | `#1AA27F` → `#3892D8` |
| 퍼플 | plum → blush | `#8B58E8` → `#EC7A9A` |
| 코럴 | coral → blush | `#F5465A` → `#EC7A9A` |

### 페이지 구성

| 경로 | 페이지 | 주요 내용 |
|------|--------|----------|
| `/` | 홈 | 히어로 슬라이더, 통계, 특징, 과목 카테고리, 프로그램, CTA |
| `/videos/:subject` | 동영상 강의 | 과목별 탭, 2×3 유튜브 그리드, 페이지네이션 |
| `/workbook/:subject` | AI 문제집 | 과목별 탭, 난이도별 문제집 카드, AI 생성기 배너 |
| `/programs` | 교육 프로그램 | 4대 프로그램 소개, 이용 단계, 과목별 바로가기 |
| `/about/:tab` | 소개 | 회사소개 / 비전 / 연혁 탭 |
| `/guide` | 이용 안내 | 이용 방법, 요금제, FAQ, 문의 |

### 제공 과목 구성

초기 5과목(수학·영어·과학·AI활용·코딩)에서 **영어·코딩 2과목**으로 정리하였다.

- 네비게이션 AI 문제집 / 동영상 강의 메뉴에 영어·코딩 서브메뉴 제공
- `subjects` 배열을 2개로 축소하여 홈 과목 카테고리 섹션에 반영
